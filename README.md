# Dalet  
Dalet is a light D Language Library for general-purpose programming.  
Current stage is not suggested for use.
Modules and functions:  
* dalet.math  
	* fact()
	* sin()
	* cos()
	* tan()
	* pow()
	* abs()
	* mults()
	* fib()
	* min()
	* max()
* dalet.array
	* sort()
* dalet.colours
	* setColour()
	* resetColour()
